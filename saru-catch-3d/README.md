# さるキャッチ！ 🐵🎈

サルゲッチュ風の、4歳児向け美しい3Dサル捕獲アクションゲームです。
iPhone / iPad の Safari(縦画面・横画面どちらでも)で遊べます。

## 遊び方

1. `index.html` をブラウザで開く(下の「起動方法」参照)
2. 緑の **▶ ボタン** をタップしてスタート
3. **行きたい場所をタップ** するとキャラクターが歩きます
4. サルに近づくと **自動であみを振って捕まえます**(ボタン操作は不要)
5. 捕まえたサルは 📖(さるずかん)にどんどん集まります

失敗・ゲームオーバー・制限時間はありません。何度でも安心して遊べます。

## 楽しい要素

| 要素 | 内容 |
|---|---|
| 🐵 サル8種類 | のんびり/ぼうし/はな/ねむり/うみ/はやて/ロケット/おうさま。それぞれ見た目も逃げ方も違う |
| 📖 さるずかん | 捕まえた種類が図鑑に登録(端末に保存)。全種類集めると🌈のおまつり |
| 🎆 ごほうび演出 | 捕まえるたび紙吹雪+ファンファーレ。5匹ごとに花火大会 |
| 🍌 バナナあつめ | 島じゅうのバナナを拾って数をためる(端末に保存) |
| 🎈 ふうせん | タップするとパーン!と割れて紙吹雪 |
| 🤸 トランポリン | 乗ると大ジャンプ。サルも跳ねる |
| 🛝 すべり台 | 丘のてっぺんから「ひゅ〜!」と滑走 |
| 🔴 大きなボタン | 踏むと花火が打ち上がる |
| ⛲ ふん水・🦋ちょうちょ・☁️流れる雲・🌸桜の木 | 島をにぎやかに演出 |
| 🎵 BGM+効果音 | WebAudioで全てコード生成(外部音源ファイル不要) |
| 😌 ゆるい段階制 | 捕まえるほどサルが少しだけ速くなる(必ず追いつける上限つき)。全24匹の後はエンドレスモード |

UIは絵文字アイコン中心で、文字が読めなくても遊べます。

## 起動方法

### かんたん(おすすめ): GitHub Pages などで公開
このフォルダをそのまま静的ホスティングに置くだけで動きます(ビルド不要)。

### ローカルで確認
```bash
cd saru-catch-3d
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

Babylon.js は `vendor/babylon.min.js` として同梱済みなのでオフラインでも動作します
(読み込めない場合は自動でCDNにフォールバックします)。

## 保護者の方へ

- 課金・広告・通信(ゲームデータ送信)は一切ありません
- 進行状況(ずかん・バナナ数)は端末の localStorage にのみ保存されます
- 🔊ボタンでいつでも消音できます
