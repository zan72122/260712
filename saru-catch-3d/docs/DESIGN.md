# さるキャッチ！ 設計資料

## 決定済み仕様(発注者承認済み)

| 項目 | 決定内容 |
|---|---|
| コア遊び | サル捕獲アクション(サルゲッチュ風) |
| マップ | 広い箱庭1マップ(半径約76mの島) |
| 技術 | Babylon.js 単一HTML・ビルド不要 |
| アート | トゥーン調カラフル |
| 操作 | タップ移動+自動捕獲(縦横両対応) |
| 追加要素 | 図鑑・ごほうび演出・ギミック全部モリモリ |
| サウンド | WebAudioでBGM+SEをコード生成 |
| UI | 文字ほぼなし(絵文字アイコン中心) |
| 難易度 | 失敗なしのゆるい段階制 |

## 対象ユーザー設計(4歳児)

- **1本指・タップのみ**で全操作が完結。ドラッグ・スワイプ・複数ボタン不要
- **失敗状態が存在しない**: 敵なし、ダメージなし、時間制限なし、ゲームオーバーなし
- サルは逃げるが**スタミナ切れで必ず立ち止まる**ため、追い続ければ100%捕まえられる
- 捕獲判定は「近づくだけ」で自動発動(あみ振りは演出として自動再生)
- UIは絵文字+ひらがな最小限。誤タップしてもすべて安全(閉じるだけで復帰)
- 効果音・演出は「うるさすぎない祝福」を基準に音量調整済み。消音ボタンあり

## ワールド構成

```
              北
        🌲森の丘(桜まじり)
   🏖砂浜 ～～～ 海 ～～～
  ⛰丘(南東)      🌼草原
        中央プラザ
   ⛲ふん水 / 🤸トランポリン(10,12)
   🔴大ボタン(-10,10) / 🎈ふうせん×6
  ⛰すべり台の丘(-40,-30)→🛝→プラザ方面
```

- 地形はガウシアン起伏3つ+海への傾斜を解析関数 `terrainH(x,z)` で表現。
  物理エンジン不使用で、全キャラのY座標は同関数から直接算出(軽量・破綻なし)
- 海面より下(`h < -0.9`)へは移動不可。うみザルのみ波打ち際までOK

## サルの種類とAI

| 種類 | 特徴 | 速度 | スタミナ |
|---|---|---|---|
| のんびりザル | 基本種 | 2.2 | 2.6s |
| ぼうしザル | 赤帽子 | 2.7 | 3.2s |
| はなザル | 花飾り | 2.5 | 3.0s |
| ねむりザル | 寝ている(💤)。近づくと起きる | 2.1 | 2.0s |
| うみザル | 波打ち際に出現、水しぶき | 2.9 | 3.4s |
| はやてザル | 最速 | 3.4 | 3.8s |
| ロケットザル | 大ジャンプで逃げる | 2.9 | 3.6s |
| おうさまザル | 金色・王冠。最後に登場 | 3.3 | 4.4s |

- AIステート: `idle → flee(プレイヤー9m以内) → tired(スタミナ切れ・💦) → idle`
- プレイヤー速度4.8 > 最速サル3.4×難易度係数上限1.24 ≈ 4.2 のため必ず追いつける

## 段階制(難易度カーブ)

- 出現キューは全24匹。簡単な種類から始まり、王冠ザルで締める
- 常時4匹が島にいる。1匹捕まえると1.6秒後に次が出現
- 難易度係数 = `1 + min(捕獲数,20) × 0.012`(最大+24%)
- 24匹捕獲後はエンドレスモード(全種ランダム+パーティー帽)

## ごほうび設計

- 毎捕獲: 紙吹雪+星バースト+4音ジングル+中央絵文字(🎉⭐💮👏ローテ)
- 5匹ごと: 花火5連発+ファンファーレ
- 図鑑コンプ(全8種発見): 花火10連発+🌈+👑
- バナナ10本ごと: キラキラ音+🍌

## サウンド実装

- `AudioContext` は最初のタップ(▶ボタン)で生成(iOS Safariの自動再生制限対応)
- BGM: 8分音符シーケンサー(132BPM・4小節ループ)。矩形波メロディ+三角波ベース+ノイズハット+サインキック
- SE: 全てオシレータ+ノイズバッファで合成(pop/boing/swish/splash/花火/ジングル/サル鳴き声など12種)

## パフォーマンス施策(iPhone/iPad想定)

- シャドウマップ不使用 → 全キャラ「ブロブシャドウ」(黒半透明ディスク)
- 花140本は thinInstance(1ドローコール)
- 雲はメッシュマージ、島の木は約26本に制限
- `setHardwareScalingLevel(1/min(devicePixelRatio,2))` で高DPI端末の負荷を制限
- 常時アクティブなサルは4匹のみ(24匹を順次出現)
- 検証環境(SwiftShaderソフトウェアレンダリング)で14〜17fps → 実機GPUでは60fps相当を期待
