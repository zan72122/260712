# 🍓 いちごましゅまろのおか

「苺ましまろ」の空気感にインスパイアされた、**4歳のお子さま向けの美しい3Dおさんぽゲーム**です。
パステルカラーのふわふわな丘で、いちごを集めたり、ましゅまろひつじと遊んだり、
夕焼けや星空をながめたり——「まけ」のない、やさしい世界です。

<p align="center">🍓 🐑 🦋 🎈 🌈 🎆</p>

ゲームオーバーはありません。

## 起動方法

ブラウザのセキュリティ制約（ES外部ファイル読み込み）のため、HTTPサーバー経由で開いてください。

```bash
cd strawberry-marshmallow-3d
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

iPhone / iPad で遊ぶ場合は、GitHub Pages などで公開して Safari で開くのがおすすめです。
「ホーム画面に追加」をするとフルスクリーンで遊べます。
**縦画面・横画面のどちらにも自動で最適化**されます。

## 3D表現のこだわり

- **カスタムシェーダーの空ドーム**: 3段グラデーション＋太陽グロウ。約4分20秒で昼→夕焼け→星空→朝焼けが一巡する昼夜サイクル（キーフレーム11点のカラーパレット補間）
- **シェーダー水面の池**: 波紋・きらめき・夜は月光色に変化。すいれんの葉がぷかぷか揺れる
- **トゥーンシェーディング**(4段グラデーションマップ)＋PCFソフトシャドウで、絵本のような質感
- **手続き生成の地形**: 重ねたサイン波によるなだらかな丘。解析関数なのでキャラも小物もぴったり地面に立つ
- **インスタンス描画**: 花畑520本・草700株・桜の花びら130枚・紙ふぶき160枚を軽量に描画
- **生きている世界**: ましゅまろ雲、いちご気球、風車、ことりの編隊、ちょうちょ、ホタル(夜)、きのこランプ(夜に点灯)、窓明かり
- **手続きアニメーション**: 走り・髪の揺れ・スカートのはずみ・ひつじのスクワッシュ&ストレッチ、すべてコードによるプロシージャル
- **音もすべて手続き生成**: WebAudioでオルゴール風BGM（ペンタトニック）、収集チャイム、ファンファーレ等を合成。外部アセットゼロ

## 技術構成

- [Three.js](https://threejs.org/) r149（`vendor/` に同梱、ビルド不要・完全オフライン動作）
- 依存パッケージ・ビルドツールなし。静的ファイルのみ
- 構成:

```
strawberry-marshmallow-3d/
├── index.html        # エントリー（タイトル画面・HUD・CSS）
├── vendor/           # three.min.js + ライセンス
├── src/
│   ├── utils.js      # 数学・テクスチャ・トゥーンマテリアル
│   ├── audio.js      # WebAudio手続き生成サウンド
│   ├── sky.js        # 空・昼夜サイクル・照明・雲・星・月
│   ├── world.js      # 地形・池・木・花・家・風車・にじ
│   ├── character.js  # ちびキャラ生成＆アニメーション
│   ├── critters.js   # ひつじ・ちょうちょ・ふうせん・ことり・いちご
│   ├── effects.js    # 花びら・キラキラ・花火・紙ふぶき・ホタル
│   ├── ui.js         # HUD（いちごバスケット・ボタン・バナー）
│   └── main.js       # ゲームループ・入力・カメラ・収集ロジック
└── docs/DESIGN.md    # デザインドキュメント
```
