# デザインドキュメント — いちごましゅまろのおか

## コンセプト

「苺ましまろ」が持つ *ゆるくて、かわいくて、なにも起こらないのに幸せ* な空気感を、
4歳児がひとりで遊べる3D空間に落とし込む。

- **敵なし・失敗なし・時間制限なし** — ネガティブなフィードバックが一切ない
- **文字が読めなくても遊べる** — 進行表示はすべて絵（いちごのバスケット）
- **さわったら、ぜんぶ反応する** — 画面に見えるキャラクターはすべてタップに反応
- **ごほうびは「うつくしさ」** — 花火・にじ・星空・夕焼けが最大の報酬

## ターゲットと操作設計（4歳児 / iPhone・iPad）

| 設計判断 | 理由 |
| --- | --- |
| ワンタップ移動（タップ地点へ自動走行） | ジョイスティックは4歳には難しい。押しっぱなしドラッグでの方向操作も併用可 |
| いちごは近づくだけで自動収集 | 「取るボタン」を無くし、偶然でも成功する |
| 当たり判定は大きめ（1.35m） | 小さな指の誤差を吸収 |
| ボタンは48px以上・画面の角に配置 | 誤タップ防止と押しやすさの両立 |
| 縦・横画面両対応 | 縦持ちは視野角60°・カメラ高め、横持ちは52°に自動調整 |
| iOSジェスチャー抑制 | `touch-action:none`・ダブルタップズーム抑制・safe-area対応 |
| 音はスタートボタンで解禁 | iOS SafariのAudioContext制約に対応 |

## ゲームループ

```
いちごを見つける → 走って拾う（チャイム＋キラキラ＋ぴょん）
    → バスケットが1つ埋まる（HUDでポップ演出）
    → 10個で「おおきなお祝い」：ファンファーレ・花火7発・紙ふぶき160枚・
       にじ出現・ひつじ全員大ジャンプ・ふたりでツイル
    → バスケットが空になり、また始まる（総数カウントは増え続ける）
```

サブループ（飽きさせない仕掛け）:
- ふうせんを割る → いちごが1個降ってくる（射的的な楽しさ）
- おともだちをタップ → 12秒に1回まで、いちごのプレゼント
- 昼夜ボタン → 世界の見た目が一変（ホタル・星・ランプ・窓明かり・月）

## ビジュアル設計

### カラーパレット
パステル基調。いちご = `#ff4d6a`〜`#ff8fb3`、草原 = `#7ecb69`〜`#b2e693`、
空（昼）= `#4db3f5`→`#fdeef4`、空（夜）= `#0a1033`→`#35407e`。

### ライティング
- キーフレーム11点（昼→夕→夜→暁）をスムーズステップ補間
- 太陽光 0.68 / 半球光 0.4 / 環境光 0.22 に調整し、トゥーンの白飛びを防止
- 夜は月光（青白い平行光 0.22）＋きのこランプのポイントライト4灯

### シェーダー
1. **空ドーム**: 視線方向の高さで3色補間 + `pow(dot(dir,sun),180)` の太陽コア + ハロー
2. **水面**: 頂点をサイン波で揺らし、中心距離ベースの波紋 + しきい値スパークル + 夜色ミックス

### パフォーマンス
- インスタンス描画: 花(頭+茎)2ドロー / 草1 / 花びら1 / 紙ふぶき1
- パーティクルは全てプール制（キラキラ420粒を使い回し）
- `pixelRatio ≤ 2`、影マップ2048、フォグで遠景カリング
- 60fps想定。`dt`は50msでクランプし、低速端末でも破綻しない

## サウンド設計（全て合成）

- **BGM**: Cメジャーペンタトニック64ステップのオルゴール。サイン波＋4倍音、
  ディレイ(0.28s)によるやわらかい残響。単純ループでも耳に痛くない音色を優先
- **収集音**: 上昇3音アルペジオ。集めるたびに開始音程が変わり単調にならない
- **ファンファーレ**: 6音スケール駆け上がり＋4和音フィニッシュ
- **効果音**: ポップ(周波数スイープ)、ぼよん(サインの往復スイープ)、きらきら(高音ベル群)

## ファイル間の依存

`utils → audio → sky → world → character → critters → effects → ui → main`
の順に読み込み。すべて `window.IM` 名前空間に登録するクラシックスクリプト構成
（ビルド不要・`file://` でも理論上動作・古いiPadのSafariでも動く）。
