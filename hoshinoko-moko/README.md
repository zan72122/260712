# ほしのこモコ 〜ふわふわしまの まいにち〜 🐱⭐

「どこでもいっしょ」のような **癒し系キャラクターふれあい 3D ゲーム** です。
空に浮かぶ小さな島で、まっしろな「ほしのこ」**モコ** と一緒にのんびり遊びます。

**4歳のお子さまが iPhone / iPad で遊ぶこと**を想定して作られています。
縦画面・横画面のどちらにも対応し、文字はすべて ひらがな、操作はすべて大きなボタンとタッチだけです。

## うごかしかた

ES Modules を使っているため、静的サーバー経由で開いてください。

```bash
cd hoshinoko-moko
python3 -m http.server 8000
# ブラウザ/iPhone/iPad で http://<マシンのIP>:8000 を開く
```

GitHub Pages などの静的ホスティングにそのまま置いても動きます。
ネットワーク接続は不要です（Three.js は `vendor/` に同梱・音はすべてリアルタイム合成）。

## みどころ（3D表現）

- **昼・夕・夜のフルサイクル**: カスタム空シェーダー（太陽・三日月・またたく星・地平線グロー）
- **UnrealBloom による発光表現**: 月のハロー、ランタン、ホタル、光るクリスタル、モコの頭の星
- **水の表現**: 疑似コースティクスときらめきの池、島から流れ落ちる滝と霧、小川
- **生きている世界**: 風にそよぐ草 900 本（GPUインスタンシング＋頂点シェーダー）、舞い散る桜吹雪、蝶、小鳥、あひる、浮遊する小島
- **ソフトシャドウ + ACES トーンマッピング** のパステル絵本ルック
- **オルゴール風 BGM**: Web Audio でリアルタイム生成（夜はテンポと音域が変わる）。効果音もすべて合成

## フォルダ構成

```
hoshinoko-moko/
├── index.html        # エントリ（importmap / HUD / スプラッシュ）
├── style.css         # UI スタイル（セーフエリア対応）
├── src/
│   ├── main.js       # ループ・入力・カメラ・モコのきもちAI
│   ├── world.js      # 島・空・水・木・家・昼夜サイクル
│   ├── moko.js       # 主役キャラクター（造形と手続きアニメ）
│   ├── effects.js    # パーティクル（ハート/桜/ホタル/流れ星/花火）
│   ├── props.js      # りんご/ボール/しゃぼんだま/プレゼント/水やり
│   ├── critters.js   # ちょうちょ・ことり
│   ├── audio.js      # Web Audio 合成（BGM & SFX）
│   └── ui.js         # HUD・トースト・スプラッシュ
├── docs/DESIGN.md    # 設計ドキュメント
└── vendor/three/     # Three.js r170（MIT License 同梱）
```
