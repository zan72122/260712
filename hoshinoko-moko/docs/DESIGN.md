# ほしのこモコ — 設計ドキュメント

## 1. コンセプト

- 参照体験: 「どこでもいっしょ 私なえほん」— キャラクターと**ふれあうこと自体が遊び**になっている
- ターゲット: 4歳児。**失敗が存在しない**。すべての操作が必ずかわいい・きれいな反応を返す
- プラットフォーム: iPhone / iPad の Safari（縦・横両対応、タッチ前提、オフライン動作）

### 4歳児向けの設計原則

1. **文字を読めなくても遊べる** — ボタンは絵文字、ヒントは補助（ひらがな）
2. **当たり判定はすべて大きめ** — モコは半径0.95の透明球、流れ星は半径4.5の透明球
3. **時間制限・スコア・ゲームオーバーなし** — ⭐カウントは「増えるだけ」
4. **すべての操作に即時のご褒美** — 音 + パーティクル + アニメーションの3点セット
5. **放置しても世界が生きている** — モコの自発行動、雲、蝶、あひる、プレゼントの定期配達

## 2. 画面と操作

タッチ優先度（`main.js` の pointerdown）:

```
流れ星 > プレゼント > しゃぼんだま > ボール > モコ > じめん(歩行) > カメラ回転
```

- ドラッグ開始位置がモコ → なでなで（ゴロゴロ音 + ハート）
- ドラッグ開始位置がボール → フリック投げ（画面ベクトル→ワールドへ変換）
- 2本指ピンチ → ズーム（範囲制限つき）
- 縦画面は FOV 66 + カメラ距離 13.5、横画面は FOV 52 + 11.5 に自動調整

## 3. ワールド構成（world.js）

浮遊島（半径7）に以下を手続き生成で配置:

| 要素 | 実装 |
|---|---|
| 空 | 半径110の球に カスタム GLSL。地平線グラデ・太陽グロー・三日月+ハロー・ハッシュベースの星（またたき） |
| 昼夜サイクル | `PHASES[3]` のパレットを毎フレーム指数補間（色/光/霧/窓明かり すべて連動） |
| 島 | 円柱+逆円錐。**位置ハッシュの決定論的ジッター**でローポリ化（亀裂防止が要点） |
| 草 | InstancedMesh 900本。`onBeforeCompile` で頂点シェーダーに風の揺れを注入 |
| 池 | 疑似コースティクス + スパークル + 縁の泡の GLSL。夜は深い藍色に |
| 滝 | スクロールする縞のアルファシェーダー + 滝つぼの霧（加算Points） |
| 桜 | 幹は曲げた円柱の積層、花冠はジッター付き二十面体×6。花びらは常時舞う |
| キノコの家 | 夜に窓とランタンが灯る（emissive + PointLight を昼夜で補間） |
| 虹 | 半トーラス + 7色バンドの GLSL。水やり・おいわいで8秒フェードイン/アウト |
| 浮遊小島/クリスタル | 島の周囲と底面。クリスタルは夜に発光が強まる（ブルームで輝く） |

## 4. モコ（moko.js）

- 全身をプリミティブ（球・カプセル・円錐・トーラス）で構成。白い体、ピンクの耳先・尻尾先、頭上に金の星
- **状態機械**: `idle / walk / eat / happy / sleep`
- 手続きアニメ: 呼吸、まばたき（確率）、しっぽ振り（ごきげん度で加速）、耳ぴく、
  ぴょこぴょこ歩き（スクワッシュ&ストレッチ）、ジャンプ+回転+着地のぺちゃ、もぐもぐ、丸くなって寝る
- 目は3モード切替: ●●（通常）/ ^^（うれしい）/ ――（睡眠）
- 頭の星は夜に PointLight となり、モコ自身をランタンのように照らす
- ぼうし4種（ニット帽/花冠/金冠/いちごベレー）をプレゼントで解放、`localStorage` に保存

### きもちAI（main.js の updateMokoAI）

1. 置かれたりんごがあれば食べに行く（最優先）
2. ボールが止まっていれば追いかけて蹴り返す（島の中心向きに補正して場外を防ぐ）
3. 夜に22秒ひまなら眠る（Zzz…、タッチで起きる、朝が来ても起きる）
4. それ以外は 7〜16秒ごとに散歩かジャンプ

## 5. エフェクト（effects.js）

- 汎用バーストパーティクル（Points + 加算合成 + 重力）を、きらきら/紙吹雪/花火/水しぶきに使い回し
- Canvas 生成テクスチャ: グロー円 / ハート / Zzz
- 流れ星: グロースプライト + 22点のライントレイル。カメラの向いている方角の低空（仰角6〜17°）に出現
- 夜はカメラの俯角を自動で 10° 浅くして星空が見えるようにする

## 6. サウンド（audio.js）

外部ファイルなし、全て Web Audio 合成:

- **BGM**: ペンタトニックのランダムウォークをオルゴール音色（基音+4倍音、指数減衰）で生成、
  フィードバックディレイで残響。夜はテンポ0.36→0.5秒、音域を1オクターブ下げ、休符を増やす
- **SFX**: にゃー（三角波の音程スイープ+ビブラート+バンドパス）、ゴロゴロ（ノイズ+23Hz振幅LFO）、
  ぽん/ぼよん/シャリシャリ/ちりん/じゃーん/水音/あくび
- 夜だけ虫の声（4.2kHz正弦波の連続パルス）
- iOS対策: AudioContext は「はじめる」ボタンのタップで生成

## 7. パフォーマンス

- devicePixelRatio を 2 にクランプ
- 草・花・花びらは InstancedMesh、ホタル・霧・雨は Points
- シャドウマップ 2048 一枚（太陽光のみ）、ポイントライトは影なし
- バースト系は使い捨てジオメトリを都度 dispose

## 8. 検証

Playwright + ヘッドレス Chromium で自動検証済み:

- 昼/夕/夜の描画（縦・横）
- ボタン5種 → りんご/ボール/しゃぼん/水やり/空切替の動作
- プレゼント開封 → ぼうし装着（`state.hat` 変化）
- 流れ星タップ → ⭐加算（`state.stars` 変化）
- 夜の放置 → 睡眠遷移
- コンソールエラー 0 件
