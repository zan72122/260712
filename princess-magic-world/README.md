## 🚀 うごかしかた

ビルド不要・外部通信なしの静的サイトです。フォルダを配信するだけ:

```bash
cd princess-magic-world
python3 -m http.server 8000
# → http://localhost:8000 を Safari / Chrome で開く
```

- **iOS 16.4 以上の Safari** / モダンブラウザ対応（ES Modules + Import Maps 使用）
- ホーム画面に追加すると全画面であそべます
- 集めた ⭐ と 💖 は `localStorage` に保存され、つぎに開いたときも残ります

### URLパラメータ

| パラメータ | 効果 |
|---|---|
| `?quality=high` | 常に最高画質（ブルーム ON・高解像度） |
| `?quality=low` | 常に軽量画質（古い端末むけ） |
| なし | 自動。カクつきを検知すると段階的に軽くなります |

## 🌟 みどころ（3Dへのこだわり）

- **UnrealBloom による発光表現** — ティアラの宝石・お城の窓・クリスタル・星が本当に光る
- **昼 / 夕焼け / 夜のフルトランジション** — 空・霧・太陽光・虹・雲・水の色まですべてが滑らかに変化
- **プロシージャル生成の空シェーダー** — グラデーションの空、太陽と月、夜はまたたく星空
- **シェーダーアニメーションの水面** — ゆらめく擬似コースティクスと波紋が広がる池と噴水
- **1,000 以上のパーティクル** — キラキラの足あと・花火・魔法の星の雨・桜の花びら・ホタル・光の粒
- **生きている世界** — はためく旗、ゆれる花、まわる小鳥、ぷかぷかのあひる、ついてくるユニコーン
- **音もすべて手続き生成** — WebAudio だけで作ったオルゴールのワルツと効果音（音声ファイル 0 個）
- **ACES トーンマッピング + ソフトシャドウ** で絵本のようなパステル映像

## 📁 フォルダ構成

```
princess-magic-world/
├── index.html          # エントリーポイント（Import Map / HUD / タイトル画面）
├── css/style.css       # UI スタイル（縦横対応・セーフエリア対応）
├── js/
│   ├── main.js         # 起動・ゲームループ・カメラ・画質自動調整
│   ├── world.js        # 地形・お城・池・木・花・虹・クリスタル
│   ├── sky.js          # 空シェーダー・昼夕夜・雲・ライティング
│   ├── princess.js     # プリンセス（モデリング + アニメーション）
│   ├── creatures.js    # ユニコーン・うさぎ・ことり・ちょうちょ・あひる
│   ├── collectibles.js # 星とハート・おいわいシステム
│   ├── particles.js    # パーティクルエンジン
│   ├── audio.js        # オルゴール音楽と効果音（WebAudio）
│   ├── input.js        # タップ操作（レイキャスト）
│   ├── ui.js           # HUD・ボタン・バナー
│   └── utils.js        # ノイズ・乱数・数学ヘルパー
├── vendor/             # Three.js r160（同梱・オフライン動作）
└── docs/DESIGN.md      # 設計ドキュメント
```
