# 設計ドキュメント — なつやすみ うみのぼうけん

## コンセプト

「ぼくのなつやすみ2 海の冒険編」の心象——
*入道雲、セミの声、きらめく海、夕焼け、ちょうちんの灯り、ほたる、花火*——を、
**4歳児がひとりで遊びきれる操作系**に落とし込む。

### 4歳児向けデザイン原則

1. **失敗がない**：釣りは逃げても「もういちど！」、時間制限なし、ゲームオーバーなし
2. **文字はぜんぶひらがな + 絵文字**、ボタンは指2本ぶんの大きさ
3. **近づくだけで拾える**（貝がら・ほたる）、**ボタン1つで完結**（つり・むしとり）
4. **すべての操作に音と光のごほうび**：チャイム、紙ふぶき、ファンファーレ、ジャンプ
5. **どの向きでも遊べる**：縦画面はFOV70/カメラ高め、横画面はFOV55

## 世界のつくり

座標系：+z が南（浜辺と海）、-z が北（丘）。島の半径はおよそ44。

```
        [北] とりい・いしどうろう（丘の上）      とうだい（東の岬）
              まるい木々・くさはら・おはな
        いえ（えんがわ・ふうりん）   おおきな木（カブトムシ）
              ひまわりばたけ（ちょうちょ・よるはほたる）
        [南] すなはま（かいがら・カニ・すいか・ボート）
              さんばし（ちょうちん・かざぐるま）→ つり場
        うみ（あさせ→ふかみ）  イルカの回遊路  花火の打上げ地点
```

### 高さ関数の共有

`G.islandHeight(x,z)` が唯一の地形定義。

- 地形メッシュの頂点変位
- キャラクターの接地・進入可否（h > -0.55 まで＝浅瀬までOK）
- 海シェーダーの浅瀬判定・岸の泡（GLSLに同式を移植）

を全部この1つの式から導くことで、砂浜と波打ち際が必ず一致する。

## 見た目（3Dへのこだわり）

- **海**：頂点シェーダーで3方向の正弦波合成。フラグメントで
  深さ→色（ターコイズ→群青）、フレネル反射（空の色）、太陽/月のスペキュラ
  きらめき（ノイズテクスチャ2枚スクロールの積）、サンロード、岸の泡（呼吸する）
- **空**：ドームシェーダー。8キーの時刻パレット補間、太陽ディスク+グロー、
  夕焼けの方向性グロー、ハッシュ星空（またたき付き）、月。仰角を1.9倍に
  圧縮して低いカメラでも天頂色が見える
- **光**：太陽方向のDirectionalLight（PCFソフトシャドウ2048px、プレイヤー追従）+
  時刻連動のHemisphereLight。夜は月光色に切替え
- **色管理**：Three.jsのカラーマネジメント（sRGB→Linear）を通した発色で
  全時刻のパレットを目視調整済み。ACESトーンマッピング
- **夜の演出**：ちょうちん・家の窓・灯台が emissive で灯り、灯台ビームが回転、
  ほたるが明滅、沖に打ち上げ花火（音つき）

## サウンド（全合成）

| 音 | 作り方 |
|---|---|
| 波 | ピンク寄りノイズ + LPF480Hz + 2つの超低速LFOで寄せ引き |
| セミ | ノイズ→BPF4.2kHz(Q9) + 42Hz振幅変調、昼だけ |
| スズムシ | 4.3kHz正弦2本のうなり + リーン…のリズムエンベロープ、夜だけ |
| 風鈴 | 高い正弦3本の減衰、ランダム間隔 |
| あたり！ | 880/660Hzの矩形波トリル + バイブレーション |
| ファンファーレ | ド・ミ・ソ・ドの三角波、大物は専用シーケンス |
| 花火 | 正弦スイープ（ヒュ〜）+ ノイズLPFスイープ（ドーン） |

## ゲームループ

`main.js` の tick：入力 → プレイヤー → あそび → 空 → 海 → 島 → いきもの →
パーティクル → 音ミックス → UI → カメラ → 描画。dt は 0.05s でクランプ。

### 釣りのステートマシン

```
idle →(ボタン)→ waiting（キャスト0.6s→ウキが波に浮く、2.2〜5.5s）
     → bite（❗表示・ウキが沈む・バイブ、3秒以内にタップ or ボタン）
        ├─ 成功 → reeling（魚が弧を描いて飛んでくる）→ おいわい → idle
        └─ 時間切れ →「もういちど！」→ waiting（短縮再抽選）
```

魚は重み付き抽選（あじ46 / たい24 / ふぐ16 / たこ10 / にじうお4）。

## パフォーマンス

- ジオメトリ：地形150², 海190²、他は低ポリのプリミティブ合成
- 草300本・花90輪は InstancedMesh
- パーティクルは固定長 Float32Array プール2本（加算/通常）、ジオメトリ再割当なし
- devicePixelRatio 上限2、シャドウマップ1枚のみ
- テクスチャは実行時にCanvas/DataTextureで生成（外部アセット0、通信0）

## 検証

Playwright + SwiftShader でヘッドレス検証済み：
タイトル→開始→歩行→貝がら収集→釣りフルフロー→むしとり→すいかわり→
昼/夕/夜の描画→ずかん表示、コンソールエラーなし（縦・横両画面）。
