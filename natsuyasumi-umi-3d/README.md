# 🏖️ なつやすみ うみのぼうけん

「ぼくのなつやすみ」シリーズにインスパイアされた、**4歳児向けの美しい3D夏休みゲーム**です。
むぎわらぼうしの男の子になって、小さな島の夏を思いっきり楽しもう！

### できること（もりもり200%）

- 🐚 **かいがらあつめ** — 浜辺できらめく貝がらを拾おう。近づくだけでOK！
- 🎣 **つり** — 桟橋の先っぽで「つりをする！」。ウキが沈んで「❗」が出たらタップ！あじ・たい・ふぐ・たこ、そして幻の「にじうお」も…
- 🦋 **むしとり** — ひまわり畑のちょうちょ、大きな木のカブトムシを虫あみでゲット
- ✨ **ほたる** — 夜になると草はらにほたるが舞う。歩いてつかまえよう
- 🍉 **すいかわり** — ビーチのすいかを3回たたくとパッカーン！
- 🦀 **カニさんダンス** — カニをタップするとばんざいダンス
- 🐬 **イルカのショー** — さかなを3びき釣ると、イルカたちが遊びに来る
- 🎆 **花火大会** — 夜は沖から打ち上げ花火
- 📖 **なつやすみずかん** — 見つけた12種類のいきものをコレクション

### 一日の流れ

朝 → 真昼の入道雲 → **息をのむ夕焼け** → 星空と月あかりの夜。
時間は自動でゆっくり流れます（1日約7分）。左上の ☀️/🌙 ボタンでワープもできます。

- 昼：セミの声、カモメ、海のきらめき
- 夕方：茜色の空と海、長い影
- 夜：ちょうちんに灯がともり、灯台が光り、スズムシが鳴き、ほたると花火

## 📱 うごかしかた

静的ファイルだけで動きます。ビルド不要・外部通信なし（Three.js も同梱）。

```bash
cd natsuyasumi-umi-3d
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
# iPhone/iPad なら同じWi-Fi内で http://<PCのIP>:8000
```

GitHub Pages などの静的ホスティングにそのまま置いてもOKです。
iPhone/iPad の Safari で「ホーム画面に追加」するとフルスクリーンで遊べます。

## 🛠️ 技術メモ

- **Three.js r158**（`vendor/` に同梱、MITライセンス）
- 海・空は**カスタムGLSLシェーダー**（うねる波・浅瀬の色・岸の泡・夕焼け・星・月・太陽のサンロード）
- サウンドは**すべてWebAudioで合成**（波・セミ・スズムシ・風鈴・効果音・ファンファーレ。音声ファイルなし）
- 昼夜サイクルは8つのキーパレットを補間
- パーティクル（しぶき・紙ふぶき・花火）は Points ベースの軽量プール
- 影あり・ACESトーンマッピング・ピクセル比上限2でモバイルでも軽快

くわしくは [docs/DESIGN.md](docs/DESIGN.md) へ。

```
├── index.html        # エントリ（タイトル画面・HUD・ずかん）

├── js/

│   ├── island.js     # 地形・ヤシ・家・桟橋・鳥居・灯台など
│   ├── character.js  # 主人公「ぼく」
│   ├── creatures.js  # カモメ・イルカ・カニ・魚影
│   ├── activities.js # つり・むしとり・かいがら・すいか・花火・ずかん
│   ├── input.js      # バーチャルスティック・タップ
│   ├── ui.js         # HUD・トースト・ずかん画面
│   └── main.js       # 起動・カメラ・メインループ
├── vendor/           # Three.js r158 (+ライセンス)
└── docs/DESIGN.md    # 設計ドキュメント
```
