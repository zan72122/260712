# きらきらプリンセス 〜おしろのまいにち〜 👸✨

プリンセスメーカー4風の、4歳児向け美しい3Dプリンセス育成ゲームです。
iPhone / iPad の Safari(縦画面・横画面どちらでも)で遊べます。

## 遊び方

1. `index.html` をブラウザで開く(下の「起動方法」参照)
2. ピンクの **▶ はじめる** をタップ
3. 下のボタンから **すきなおけいこ** をタップすると、プリンセスが歩いて行って遊びます
4. 遊ぶたびに ⭐ と 💗 がもらえて、たまると **レベルアップ🎉**
5. レベルが上がると 👗 おきがえアイテム がどんどんアンロック!

失敗・ゲームオーバー・制限時間はありません。何度でも安心して遊べます。

## おけいこ(6種類)

| ボタン | 内容 |
|---|---|
| 🍰 おちゃかい | ケーキをもぐもぐ。ハートがぽんぽん飛び出す |
| 💃 ダンス | ガゼボ前のステージでくるくるツィンクルダンス |
| 🎨 おえかき | イーゼルのキャンバスに**本当に虹の絵が描かれていく** |
| 🪄 まほう | 泉に向かって魔法の練習。キラキラが吹き上がる |
| 🐴 ポニー | ポニーに乗ってお庭をぐるぐるお散歩(カメラも追いかける) |
| 🌷 おはな | 花壇にお花を植える。植えた花は**ずっと残って花畑になる** |

## モリモリの楽しい要素

- 👗 **きせかえ26種類** — ドレス8着(レインボー・星空ドレスも!)、髪型4種、髪色5色、頭飾り7種(ティアラ・王冠・妖精の羽・天使の輪)。レベルで順番にアンロック
- 👑 **レベルアップ演出** — ファンファーレ+花火+虹+紙吹雪。プリンセスも少しずつ大きく成長
- 💃 **舞踏会イベント** — 4回遊ぶたびにスポットライトの当たる特別なダンスタイム
- 🌅 **昼夜サイクル(1日150秒)** — 昼→息をのむ夕焼け→星空の夜→朝焼け。夜はお城の窓・ランタン・ガゼボのイルミが灯り、ホタルが舞う
- 🌸 **生きているお庭** — 舞い散る桜の花びら、ひらひら飛ぶちょうちょ、流れる雲、はためく旗、きらめく噴水
- 🏰 **パステルピンクのメルヘン城** と虹のお祝い演出
- 🚶 **地面をタップするとお散歩** — 好きな場所に歩いて行ける。小物をタップするとキラキラ
- 🎵 **オルゴール風ワルツBGM+30種の効果音** — WebAudioで全てコード生成(外部音源ファイル不要)。夜はゆっくり子守唄テンポに
- 💾 **自動セーブ** — スター・レベル・きせかえ・植えた花は端末に保存

UIは絵文字アイコン中心で、文字が読めなくても遊べます。

## 起動方法

### かんたん(おすすめ): GitHub Pages などで公開
このフォルダをそのまま静的ホスティングに置くだけで動きます(ビルド不要)。

### ローカルで確認
```bash
cd princess-maker-3d
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

Babylon.js は `vendor/babylon.min.js` として同梱済みなのでオフラインでも動作します
(読み込めない場合は自動でCDNにフォールバックします)。

## 構成

```
princess-maker-3d/
├── index.html        # 画面・UI・CSS
├── js/
│   ├── save.js       # セーブデータ(localStorage)
│   ├── audio.js      # BGM・効果音(WebAudioで完全生成)
│   ├── world.js      # 3Dワールド(空・城・庭・噴水・パーティクル)
│   ├── princess.js   # プリンセス&ポニー(プロシージャルモデル+きせかえ)
│   ├── activities.js # おけいこ・舞踏会・レベルアップ進行
│   ├── ui.js         # HUD・おきがえパネル・お祝い演出
│   └── main.js       # エンジン起動・カメラ・入力
├── vendor/
│   └── babylon.min.js  # Babylon.js 9.16.1(同梱)
└── docs/
    └── DESIGN.md     # 設計メモ
```

## 技術メモ

- Babylon.js 9 + WebGL2。3Dモデル・テクスチャ・音源はすべてコードで生成(外部アセット0)
- ACESトーンマッピング+ブルーム+ビネットの後処理で「絵本のような」画づくり
- 影はブラー付きシャドウマップ+接地用ブロブシャドウの二段構え
- 静的メッシュはマージしてドローコール削減、Retinaは2xまでにクランプしてモバイルでも軽快
