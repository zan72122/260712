# 設計メモ — きらきらプリンセス 〜おしろのまいにち〜

## コンセプト

**プリンセスメーカー4風の「育成」体験を、4歳児が一人で遊べる形に翻訳する。**

- 原作の「スケジュールを組んで娘を育てる」→「好きなおけいこボタンをタップする」に単純化
- 原作の「パラメータ育成」→ ⭐スターの蓄積によるレベルアップ(全ステータスは裏で記録)
- 原作の「衣装・イベント」→ きせかえアンロックと舞踏会イベント
- **失敗状態を一切作らない**。すべての操作が必ずポジティブな結果につながる

## 対象年齢(4歳)のためのデザイン原則

1. **文字を読まなくても遊べる** — UIは絵文字中心、ラベルは補助
2. **タッチターゲット 52px 以上**、誤操作してもすべて安全
3. **待たせない** — おけいこ中もキラキラ・効果音が続き、必ず10秒以内にごほうび
4. **カメラで迷子にならない** — パン禁止、ズーム・角度は制限付き、プリンセスを常に自動追尾
5. **音が出なくても遊べる**(サイレントスイッチ考慮)、音はごほうびの強化として設計

## 画面・入力

- 縦画面・横画面両対応(flexレイアウト+safe-area-inset)
- ArcRotateCamera: ドラッグで回転、ピンチでズーム(制限付き)。タップとドラッグは移動量20pxで判別
- 地面タップ→自由散歩、ボタンタップ→おけいこ、メッシュタップ→キラキラ

## ゲームループ

```
おけいこ(6種) → ⭐+💗 → 5⭐ごとにレベルアップ(最大10)
                        → 4回ごとに舞踏会イベント
レベル → きせかえアンロック(26種) → 見た目の変化がさらに遊ぶ動機に
🌷だけは永続的な世界の変化(花畑が育つ)を作る
```

レベルアップと舞踏会は 5 と 4 で互いに素なので交互にリズムよく発生する。

## 3D表現(こだわりポイント)

- **完全プロシージャル**: メッシュ・テクスチャ・音すべてコード生成。アセットロード0、オフライン動作
- **空**: カスタムシェーダのグラデーションドーム。8キーフレームの昼夜サイクル(150秒/日)を色補間
- **夜**: 星140個(マージ済み1メッシュ)、月、ホタルパーティクル、窓・ランタン・イルミの発光をnightFactorで連動
- **後処理**: ACESトーンマップ+ブルーム+ビネット+FXAA(DefaultRenderingPipeline)
- **キャラクター**: 2.5頭身。まばたき・髪の揺れ・スカートの広がりまで手続きアニメーション
- **パフォーマンス**: 静的メッシュはMergeMeshes、花はclone、ハードウェアスケーリングは2xまで。
  ロジックはdt基準(0.1sキャップ)なので低fps端末でも破綻しない

## 音(WebAudio)

- オルゴール風の3拍子ワルツ(C-G-Am-F進行)をsetTimeoutで1小節ずつスケジュール
- 夜は自動でテンポと音量が下がり子守唄化
- 効果音は正弦波ベル・三角波ピッチベンド・整形ノイズの3素材から30種生成
- コンプレッサーで音割れ防止。初回タップで AudioContext を resume(iOS対策)
