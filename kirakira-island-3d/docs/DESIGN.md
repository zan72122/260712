# 設計メモ — きらきらしまの だいぼうけん

## コンセプト

Disney Illusion Island の「あたたかい絵本世界・協力・探検」の気持ちよさを、
**4歳児がひとりで遊びきれる 3D 箱庭**に翻訳する。

- **失敗が存在しない**: ダメージ・敵・時間制限・ゲームオーバーなし。
  海に落ちる= ぷかぷか泳ぐ演出、放置すれば島へ自動帰還。
- **すべての操作にごほうび**: 歩くだけでピクシーダストが舞い、
  ジャンプで白い輪がはじけ、星を取るたび音程が上がるチャイム。
- **文字が読めなくても遊べる**: 目標は「光っているものに近づく」だけ。
  友だちは遠くからピンクの光の柱で見える。バナーはひらがな+絵文字。

## 技術構成

| 領域 | 選択 | 理由 |
|---|---|---|
| 3D エンジン | Three.js r147 (UMD 同梱) | `file://` でも動く・iOS Safari 安定・オフライン |
| 地形 | 解析的ハイトフィールド `groundHeight(x,z)` | メッシュと物理が同じ関数を共有 → 衝突判定が O(1) |
| 足場 (橋・浮島・きのこ・回転木馬) | `K.platforms[]` に `heightAt(x,z)` を登録 | レイキャスト不要の超軽量プラットフォーム物理 |
| 空・海・滝 | カスタム GLSL シェーダー | 昼夜サイクルと色が完全に同期、頂点数最小 |
| 草花 | InstancedMesh (草 700 + 花 420) | ドローコール 3 で草原を実現 |
| パーティクル | 自作 CPU プール (Points × 3 + Sprite) | 花火 90 発同時でも 60fps を狙える固定コスト |
| 音 | WebAudio 完全プロシージャル | 音源ファイル 0 バイト、初回タップで解禁(iOS 制約) |
| 入力 | フローティング仮想スティック | 幼児は「決まった場所」を押せない → 触れた場所に出す |

## 物理

- 重力 + 垂直速度のみの単純な運動。水平は速度直結(慣性なし)で
  幼児でも狙った場所に止まれる。
- 接地: `support = max(terrain, 高さが足元+0.7 以下の platform)`。
  落下中に support を下回ったらスナップ。
- きのこ: 空中から着地した瞬間だけ `SUPER_JUMP` — 歩いて乗っても
  暴発しない。
- 泳ぎ: 地形高が -0.6 以下なら水面 (-0.25) にダンプ補間 + 波バブ。
  スティック無入力なら毎秒 2.2m で島中心へドリフト(自動レスキュー)。

## 昼夜サイクル (DAY_LENGTH = 150s)

`phase = 経過時間 / 150 % 1` を太陽仰角 `sin(2π·phase)` に変換し、
day / sunset / night の 3 グレードを滑らかにブレンド:

- 空ドーム: top/horizon 2 色 + 太陽ディスク & ハロー(フラグメント内)
- ライト: 平行光(影 2048px, プレイヤー追従カメラ) + 半球光 + 環境光
- 霧: 色は地平線色と同一、夜は near/far を引き寄せて灯りを際立たせる
- 夜だけ現れるもの: 星 700・流れ星・ほたる 34・灯台ビーム・月光の海きらめき

## 「面白さ200%」チェックリスト

収集(星60+コンボ音階) / 発見(友だち5) / 追従パレード / サプライズ(プレゼント4)
/ 大ジャンプ(きのこ5) / 乗り物(回転木馬) / ランドマーク踏破(にじの橋→浮島)
/ 天体ショー(夕焼け・星空・流れ星) / 生き物観察(蝶・鳥・魚・ほたる)
/ 祝祭(10個ごとの花火、全員発見で花火大会+風船+紙ふぶき) / 無限おかわり(星再出現)

## テスト

Playwright + SwiftShader でヘッドレス検証済み:
コンソールエラー 0、縦・横両画面レンダリング、橋・浮島・海・バウンスの
物理プローブ、朝・夕焼け・夜の描画確認。`KIRA._debug.teleport(x,z)` と
`?time=` パラメータはこの検証用に残してある(隠し機能として無害)。
