# ✨ きらきらしまの だいぼうけん (KiraKira Island Adventure)

「Disney Illusion Island」風の、やわらかくて美しい 3D アドベンチャー。
うさぎの **ミミ** になって、きらきら光る島を駆けまわり、おほしさまを集めて、
かくれんぼ中の 5 ひきのおともだちを見つけよう！

**対象: 4歳児 / iPhone・iPad(縦画面・横画面 両対応) / ブラウザだけで動作**

## 🎮 あそびかた

| そうさ | どうなる |
|---|---|
| 画面ひだり半分をタッチしてスライド | ミミが歩く・走る(タッチした場所にジョイスティックが出ます) |
| みぎ下の みどりのボタン(または画面みぎ半分をタップ) | ジャンプ！ |
| キーボード(PC) | 矢印キー / WASD で移動、スペースでジャンプ |

- ⭐ **おほしさま**にさわると集まるよ(ぜんぶ集めると「おかわり」が降ってくる)
- 🐻🐧🐱🐤🐷 **おともだち**を見つけると、うしろをついてきてパレードに!
  5 ひき全員見つけると**花火大会**がはじまるよ
- 🎁 **プレゼントばこ**をあけると、紙ふぶきと風船とボーナススター!
- 🍄 **きのこトランポリン**に乗ると、おそらまで大ジャンプ!
- 🌈 **にじの橋**をわたると、そらに浮かぶ島へ行けるよ
- 🎠 まわる**メリーゴーランド**に乗ってひとやすみ
- まけ・ダメージ・ゲームオーバーは**いっさいなし**。海に落ちても
  ぷかぷか泳いで、ほうっておくと島まで流れて戻ります

## 🚀 起動方法

ビルド不要・完全オフライン動作(Three.js 同梱)。

```bash
# そのまま index.html をブラウザで開く、または:
cd kirakira-island-3d
python3 -m http.server 8000
# → http://localhost:8000 を iPhone/iPad の Safari で開く
```

iPhone / iPad では「ホーム画面に追加」するとフルスクリーンで遊べます。

### 隠しパラメータ

URL に `?time=` を付けると、はじまる時間帯を選べます:
`?time=asa`(朝) / `?time=hiru`(昼) / `?time=yuyake`(夕焼け) / `?time=yoru`(夜)

## 🌅 みどころ (3Dへのこだわり)

- **一日のうつろい**: 朝焼け → まひる → 黄金の夕焼け → 星空、と約 2 分半で
  空・光・霧・海の色がシームレスに変化するカスタムシェーダーの空
- **生きている海**: うねる波、岸辺のあわ、太陽のきらめき、夜は月光の道
- **降る夜空**: 700 個のまたたく星、流れ星、ほたる、灯台のサーチライト
- **そよぐ森**: 風にゆれる木々、舞い散る桜の花びら、羽ばたく蝶、
  まわる鳥たち、海からジャンプするお魚
- **きらきらエンジン**: ピクシーダスト、紙ふぶき、ハート、花火、風船 —
  すべてプロシージャル生成のパーティクル
- **音楽も生成**: オルゴール調の BGM と効果音を WebAudio でリアルタイム合成
  (音声ファイルなし・オフライン OK)
