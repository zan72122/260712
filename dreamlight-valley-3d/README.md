## 動かしかた

ビルド不要・外部通信ゼロ（ライブラリ同梱）です。

```bash
# リポジトリを取得して
cd dreamlight-valley-3d

# そのまま index.html をブラウザで開くか、ローカルサーバーで：
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

### iPhone / iPad で遊ぶとき
1. Safari で開く
2. 共有ボタン → **「ホーム画面に追加」** するとフルスクリーンで遊べます
3. 音はいちど画面をタップすると鳴りはじめます（iOS の仕様です）

## 技術ハイライト

- **Three.js r134**（`vendor/` に同梱、CDN 不要・オフライン動作）
- **UnrealBloomPass + ACES トーンマッピング + ガンマ補正**の3段ポストプロセス
- 地形・空・水面・滝・光るキノコは**カスタム GLSL シェーダー**
- 島・川・道・木々はすべて**プロシージャル生成**（シード固定で毎回同じ地形）
- 草 2,600 本・花 600 輪は **InstancedMesh + 風揺れ頂点シェーダー**
- パーティクル：きらめき／ハート／花びら／ホタル／ミスト／打ち上げ花火（プール式 Points）
- BGM・効果音は **Web Audio でリアルタイム合成**（オルゴール調ペンタトニック生成曲、音声ファイルなし）
- 昼夜サイクル、カメラのシネマティック演出、カメラ前の草木をディザで透かす視界確保
- FPS を監視して解像度・ブルーム・影を自動で落とす**オートクオリティ**
- `localStorage` への自動セーブ
