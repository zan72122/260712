# きらきら ドリームバレー — 設計ドキュメント

## 1. コンセプト

「ディズニー ドリームライトバレー」の *コージーで魔法にあふれた谷* の雰囲気を、
**4歳児がひとりで遊びきれる** ミニマルな操作系に落とし込む。

### 4歳児向けデザイン原則
| 原則 | 実装 |
| --- | --- |
| 失敗が存在しない | 敵・体力・時間制限なし。深い水には自動で入らないだけ |
| 読めなくても遊べる | UI は絵文字とひらがな。クエストは「7つの光る柱」という視覚言語 |
| 何をさわっても反応する | 地面・水・空・木・動物・自分、全タップに音と光のフィードバック |
| ワンフィンガー操作 | タップ移動＋なぞり移動。ピンチズームのみ2本指 |
| 縦横どちらでも | 画面比でFOVを自動調整（縦62°/横50°）、セーフエリア対応 |
| 続きから遊べる | localStorage に自動セーブ（星・りんご・ともだち・にじ） |

## 2. ゲームループ

```
さんぽ ─┬─ ⭐ 星あつめ（無限に湧く・10個ごとに ほめる演出）
        ├─ 🍎 木をゆらす → りんご → 🐰 どうぶつに あげる → 💛 ともだち（追従）
        ├─ 🌈 にじのかけら×7 → シネマティックカメラ＋虹＋花火＋👑
        └─ 🌙 昼夜トグル → ホタル・光るキノコ・流れ星・（クリア後）夜の花火
```

報酬はすべて「もっときれいになる」方向に設計（虹・王冠・夜花火・行列になる動物たち）。

## 3. ワールド

半径約95の島。`terrainHeight(x,z)` （sin/cos の重ね合わせ＋ガウス丘＋距離減衰）が
**唯一の真実**で、描画メッシュ・キャラ接地・レイキャストすべてがこの解析関数を参照する
（メッシュへのレイキャスト不要 = BVH なしで高速）。

- **村**（南西）: きのこ屋根の家×3、桜、広場
- **川**: 北の湖から南の海へ `riverX(z)` 曲線で蛇行。水深 -1.0 で歩いて渡れる（水しぶき）
- **橋**: アーチ橋。`bridgeHeight()` が接地高さを上書きして上を歩ける
- **湖と滝**（北西）: 岩山からスクロールシェーダーの滝＋ミスト粒子＋距離連動の環境音
- **お城**（南東の丘）: 塔5本・金の飾り・旗アニメ・夜に光る窓
- **キノコの森**（東）: 夜に脈動発光するカスタムシェーダーのカサ
- **にじのかけら×7**: 谷全体に分散配置。上空への光の柱で遠くから視認可能

植生・岩の配置はシード付き乱数（mulberry32）で決定論的。道・川・湖・急斜面を避けて散布。

## 4. レンダリングパイプライン

```
Scene (Lambert + custom ShaderMaterial, linear color space)
  → RenderPass (ACES tone mapping / PCFSoft shadow 2048px)
  → UnrealBloomPass (strength 0.42 昼 / 0.7 夜, threshold 0.88)
  → GammaCorrectionShader (linear → sRGB)
```

- 色は `Color.convertSRGBToLinear()` で入力し、最終段で一括ガンマ補正（r134 の
  EffectComposer は outputEncoding を適用しないため）
- 空: 頂点球シェーダー。昼夜グラデーション・太陽ハロー・月光・ハッシュ星空・天の川
- 水: 1枚の大平面。頂点波＋近似法線でフレネル、太陽スペキュラ、ラメ、昼夜で色相変化
- 植生: InstancedMesh ＋共通シェーダー。インスタンス色・位相属性、風揺れ、
  擬似ランバート照明、**カメラ近傍ディザディゾルブ**（視界がふさがらない）
- 昼夜: `uDay` 0..1 を全シェーダー・ライト・フォグ・露出・ブルーム強度へ一括補間

## 5. パーティクル

固定容量プール（Points ＋ Dynamic BufferAttribute）を6系統:
sparkle / glow / heart / petal / star / mist。
重力・抗力・ゆらぎ・点滅・成長をパラメタ化し、
花火（ロケット＋爆発70粒）、ホタル130匹（プール内常駐）、桜吹雪、滝ミストを共有実装で駆動。

## 6. オーディオ（Web Audio 完全合成）

- **BGM**: 3/4拍子のオルゴール子守唄をリアルタイム生成。
  ペンタトニックのランダムウォーク旋律（倍音合成ベル）＋ハープアルペジオ＋
  デチューンパッド、フィードバックディレイで残響。夜は音数が減る
- **環境音**: ループノイズ2系統（滝=バンドパス: プレイヤー距離でゲイン連動、そよ風=LFO付きローパス）
- **SFX**: 収集チャイム・ぽん・魔法アルペジオ＋シマー・動物の鳴き声（種でピッチ差）・
  ファンファーレ・花火（ひゅ〜＋ノイズ爆発）など12種
- iOS対策: 初回タップで AudioContext を生成/resume

## 7. パフォーマンス

- 草・花・木・岩は InstancedMesh（合計 約4,000 インスタンスを十数ドローコール）
- devicePixelRatio 上限2。FPS の指数移動平均を監視し
  27fps 未満で解像度 0.7 倍 → さらに低ければブルーム＋影を停止
- 地形レイキャストは解析関数のレイマーチ（タップ時のみ、二分法14回で収束）
- パーティクルは容量固定・GC フリー

## 8. ファイル依存関係

```
audio.js ──┐
world.js ──┼── main.js（初期化順: world → FX → creatures → loop）
effects.js ┤      ↑ DV.* グローバル名前空間で疎結合
creatures.js ┘
```

## 9. 手動テスト手順（ヘッドレス済み項目）

1. タイトル→タップ→イントロカメラ→操作可能
2. 星に近づく→カウント増・チャイム
3. 木タップ→りんご落下→拾う→動物に接近→ともだち化＋追従
4. かけら7個→シネマティック＋虹＋花火＋王冠
5. 🌙→暗転・ホタル・窓明かり・ランタン点灯、☀️で復帰
6. 縦持ち/横持ち切替、ピンチズーム
7. リロード→セーブが復元される
